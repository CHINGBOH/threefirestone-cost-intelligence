# **App Name**: Tech Deep Dive

## Core Features:

- Truth About Computers Intro: Show high-tech computing scenes and AI models to draw viewers in.
- Core Revelation: The Computer Isn't Smart: Show a glowing 0 and 1 binary stream. Announce that it is not 'smart,' only a large network of switches.
- Simile for Abstraction: Light Switches: An animation of transistors being flipped to open/close circuits.
- Simile for Abstraction: Ordering Food: Like in a restaurant, your natural language request goes to a waiter who translates to the chef.
- Metaphor: the CPU Doesn't Directly Control the Device!: You press a button, and the light comes on. Did the CPU flip the switch? Explore an analogous concept of CPU, device controllers, and hardware devices, with each level receiving an abstraction of your direction, ultimately translated to binary to trigger the switch.
- Translation Between Layers of Abstraction Tool: AI uses tool to present computer functionality as metaphor between several real-world abstractions to support the comprehension of its operations and constraints.

## Style Guidelines:

- Primary color: Saturated blue (#29ABE2) for a tech-focused but friendly look.
- Background color: Very light blue (#E0F7FA).
- Accent color: Slightly darker teal (#29D0D0) to draw attention.
- Headline font: 'Poppins', a geometric sans-serif for its contemporary and fashionable look.
- Body font: 'Inter', a grotesque-style sans-serif with a modern look.
- Use high-tech icons and graphics for easy understanding.
- Subtle animations for a high-tech feel.
