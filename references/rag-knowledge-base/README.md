# RAG-knowledge-base

> RAG Knowledge Base — 基于 Next.js + Google Genkit AI + Gemini/DeepSeek 的 RAG 知识库系统。支持多模型检索增强生成。

## Tech Stack

Next.js · React · TypeScript · Tailwind CSS · Google Genkit AI · Gemini · DeepSeek · shadcn/ui

## Project Structure

| `src/ai/` | Gemini/DeepSeek 模型集成 |
| `src/app/` | Next.js App Router 知识库页面 |
| `src/components/` | shadcn/ui 组件和知识图谱 UI |
| `src/hooks/` | React Hooks |
| `src/lib/` | RAG Pipeline 工具函数 |

## Quick Start

```bash
npm install
npm run dev
# Open http://localhost:3000
# Configure Gemini/DeepSeek API keys in .env.local
```
